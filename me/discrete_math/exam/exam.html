I. Множества

1. Понятие множества. Множества и подмножества. Способы задания множества

Георг Кантор опредял множество как «многое, мыслимое как единое целое». Можно дать и немного другое определение множества: «сущность, содержащаяя другие сущность произвольной природы, объедиенные общим характеристическим свойством».

Подмножество \(B\) множества \(A\) — это такое множество, что любой элемент, принадлежащий множеству \(B\) также принадлежит множеству \(A\).

Есть несколько способов задания множества:

\[ A = \{1,\; 2,\; 3,\; 4,\; 5,\; \ldots\} \]

С помощью прямого перечисления элементов.

\[ A = \{a \;\colon\; \chi_A(a) = 1\} \]

С помощью характеристической функции.

\[ A = \{a \;\colon\; a > 0 \;and\; a < 100 \} \]

С помощью характеристического предиката.

\[ A = \{a \;\colon\; a_1 = 1,\; a_2 = 1,\; a_i = a_{i-1} + a_{i-2} \; for \; i = 2,\; 3,\; \ldots\} \]

С помощью порождающей процедуры.
